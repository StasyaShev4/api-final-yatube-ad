## Описание проекта

API для социальной сети Yatube.  
Проект позволяет пользователям публиковать посты, оставлять комментарии, подписываться на других пользователей и просматривать группы.

API реализован с использованием Django REST Framework и поддерживает аутентификацию с помощью JWT-токенов.

---

## Установка

Клонировать репозиторий:

```
git clone <ссылка_на_репозиторий>
```

Перейти в папку проекта:

```
cd api-final-yatube-ad
```

Создать виртуальное окружение:

```
python -m venv venv
```

Активировать:

```
# Windows
venv\Scripts\activate

# Linux / MacOS
source venv/bin/activate
```

Установить зависимости:

```
pip install -r requirements.txt
```

Применить миграции:

```
python manage.py migrate
```

Запустить сервер:

```
python manage.py runserver
```

## Примеры запросов

### Получить список постов

```
GET /api/v1/posts/
```

---

### Создать пост

```
POST /api/v1/posts/
```

```
{
  "text": "Мой первый пост"
}
```

---

### Получить комментарии к посту

```
GET /api/v1/posts/{post_id}/comments/
```

---

### Добавить комментарий

```
POST /api/v1/posts/{post_id}/comments/
```

```
{
  "text": "Комментарий"
}
```

---

### Подписаться на пользователя

```
POST /api/v1/follow/
```

```
{
  "following": "username"
}
```

---

### Получить подписки

```
GET /api/v1/follow/
```
